## Install

Clone the repo using

```
$ git clone https://github.com/HarrisonHenri/rentx
```

```
$ cd rentx
```

Install all the required dependencies

```
$ yarn
```

Runs the [api](https://github.com/HarrisonHenri/rentxapi)

Then execute

```
$ yarn start
```
